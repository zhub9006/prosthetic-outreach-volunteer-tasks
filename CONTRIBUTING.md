# Contributing to Prosthetic Outreach Volunteer Tasks

## Task Labels

- `clinical-research` — Clinical trial identification & analysis
- `volunteer-recruitment` — Recruiting and onboarding volunteers
- `logistics` — Supply chain, shipping, and inventory
- `community-outreach` — Local events, partnerships, awareness
- `data-analytics` — Outcome tracking and impact measurement
- `documentation` — Guides, templates, and process docs

## Workflow

1. Pick an issue from the board and claim it
2. Create a branch named `task/<issue-number>-<short-title>`
3. Make your changes
4. Open a PR referencing the issue
5. Get review and merge

## Code of Conduct

Be respectful, collaborative, and focused on our mission of expanding prosthetic access.
