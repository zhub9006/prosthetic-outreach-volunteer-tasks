# Prosthetic Outreach Volunteer Task Coordination

Coordination hub for volunteer tasks across outreach teams — tracking assignments, progress, and deliverables for the prosthetic outreach initiative.

## Mission

To coordinate volunteer efforts across multiple teams supporting affordable prosthetic solutions outreach in under-resourced communities.

## Task Workflow

1. **Pick a task** — Browse open issues labeled by team
2. **Claim it** — Comment on the issue to claim
3. **Work** — Create a branch `task/<issue-number>-<short-title>`
4. **Review** — Open a PR referencing the issue
5. **Done** — Merge after approval

## How to Use This Repo

- **Issues** — Create and track individual tasks. Use labels (`clinical-research`, `volunteer-recruitment`, `logistics`, `community-outreach`, `data-analytics`, `documentation`)
- **Project Board** — Visualize status: To Do → In Progress → Review → Done
- **Pull Requests** — All work goes through PRs for review
